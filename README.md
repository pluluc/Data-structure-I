# Data-structure-I
Programs that simulate practical situations using linked lists, queues, stacks and binary trees.   

Todos os códigos aqui presentes foram feitos para avaliações na matéria Estrutura de Dados I, onde recebemos um contexto do professor e criamos sistemas que atendiam a demanda e utilizavam de quatro tipos de estruturas de dados:

Linked list: um sofware que registra veículos e os registra em ordem alfabética, remove veículos e realiza busca por ano e por preço.

Queue: 
      Um sistema com filas de atendimento de um hospital, com prioridades;
      Registra pacientes, os insere nas filas correspondentes, chama o próximo paciente com base na ordem de prioridade: emergências primeiro e para cada 3 urgentes, um regular;
      Busca por cpf e consulta de quantos estão na frente de determinado sujeito;
      Relatório de atendimentos.
      
Binary tree:
      Um sistema de transporte aéreo com duas árvores, uma para cada país onde a empresa atua;
      Registra as passagens e gera um rastreador aleatório e único para cada uma;
      Realiza busca por destino;
      Oferece relatório;
      Organiza as vendas das duas árvores em uma terceira e limpa as duas primeiras.
